Tentang Project

Website ini dirancang untuk memprediksi harga rumah berdasarkan karakteristik properti seperti luas bangunan, luas tanah, jumlah kamar tidur, kamar mandi, dan jumlah lantai.

Project ini menggunakan pendekatan Machine Learning dengan algoritma Support Vector Regression (SVR) yang dilatih menggunakan data properti aktual. Seluruh data telah melalui proses praproses dan feature engineering untuk menghasilkan estimasi harga yang mendekati nilai pasar.

Project ini dikembangkan oleh Ulfa R sebagai tugas mata kuliah Proyek Data Mining..
